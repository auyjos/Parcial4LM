## MÁQUINA DISEÑADA

Propósito: Verificar si una cadena binaria tiene número PAR de 1's

Estados:
- q0: contador par (estado inicial)
- q1: contador impar
- qloop: ciclo infinito
- qaccept: aceptación
- qreject: rechazo

Alfabetos:
- Σ = {0, 1}
- Γ = {0, 1, x, _}

Transiciones clave:
- δ(q0, 1) = (q1, 1, R)  // Par → Impar
- δ(q1, 1) = (q0, 1, R)  // Impar → Par
- δ(q0, _) = (qaccept, _, R)  // Par → ACEPTA
- δ(q1, _) = (qreject, _, R)  // Impar → RECHAZA
- δ(q0, x) = (qloop, x, R)  // Cualquier → CICLO
- δ(q1, x) = (qloop, x, R)  // Cualquier → CICLO

---

## CASOS DE PRUEBA

Caso 1: ACEPTACIÓN
Entrada: 0110 (2 unos = par)
Resultado: ACEPTA
Configuraciones: 6 pasos

Caso 2: RECHAZO
Entrada: 001 (1 uno = impar)
Resultado: RECHAZA
Configuraciones: 5 pasos

Caso 3: CICLO INFINITO
Entrada: 01x (contiene 'x')
Resultado: CICLO_INFINITO
Configuraciones: 10,000 pasos (límite)

---

## NOTACIÓN

Configuración: uqv
- u: parte leída (izquierda)
- q: estado actual
- v: parte por leer (derecha)

Ejemplo: 011q00
- Ya leído: "011"
- Estado: q0
- Por leer: "0"

Transición: δ(q, a) = (p, b, D)
- q: estado actual
- a: símbolo leído
- p: nuevo estado
- b: símbolo escrito
- D: dirección (L/R)
